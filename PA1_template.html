<h4>2. Devise a strategy for filling in all of the missing values in the dataset</h4>
<p>As the number of missing values in this dataset is fairly large, we cannot be sure if there is no bias introduced by missing values. Therefore we impute missing values based on average number of steps in particular 5-minutes interval. </p>
<h4>3. Create a new dataset that is equal to the original dataset but with the missing data filled in.</h4>
<pre><code class="r">without_NAs &lt;- numeric(nrow(activity))
for (i in 1:nrow(activity))
{
if (is.na(activity[i,&quot;steps&quot;])==TRUE)
{
without_NAs[i]&lt;-filter(daily_patterns,interval==activity[i,&quot;interval&quot;]) %&gt;% select(average)
}
else
{
without_NAs[i]&lt;-activity[i,&quot;steps&quot;]
}
}
activity_without_NAs&lt;-mutate(activity,steps_no_NAs=without_NAs)
